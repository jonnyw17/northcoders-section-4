Coding Challenge #2 - Underscore Methods

For this challenge you'll be implementing your own version of some of Underscore's methods. Underscore is a JavaScript library with a lot of really useful utility functions to work mainly with objects and arrays. You should implement:

  • first
  • rest
  • contains
  • compact

We recommend spending some time getting familiar with what the different methods do. Read the documentation (linked above) and play with the examples in the console or write your own. We've included the actual Underscore in this project, you can access it through the "_" object both in the console and "main.js". It goes without saying that you should not use any Underscore method in your solution.
